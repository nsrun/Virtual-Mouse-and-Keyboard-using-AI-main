
# Virtual Mouse and Keyboard using AI

This project enables control of the mouse and keyboard using AI techniques, allowing for hands-free interaction with your computer. It leverages computer vision and machine learning models to interpret user gestures or inputs and translate them into mouse and keyboard actions.

## Features

- AI-powered gesture recognition for mouse control
- Keyboard input simulation
- Real-time processing
- Cross-platform compatibility

## Prerequisites

- Python 3.x
- OpenCV
- TensorFlow or other ML frameworks used
- Additional dependencies listed in `requirements.txt`

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Virtual-Mouse-and-Keyboard-using-AI-main.git
cd Virtual-Mouse-and-Keyboard-using-AI-main
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the main script:

```bash
python main.py
```

Follow on-screen instructions to calibrate and start controlling your mouse and keyboard with gestures.
